Pipes | Date | Fecha

Aca se muestra las formas de personalizar una fecha utilizando pipes.

  • Fecha
  • Fecha | date
  • Fecha date:'short'
  • Fecha date:'long'
  • Fecha date:'personalizado-números'
  • Fecha date:'personalizado-texto'
  • Fecha date:'long':'timezone'
  • Fecha date:'long':'timezone':'locale'
  • {{ fecha }}
  • {{ fecha | date }}
  • {{ fecha | date: "short" }}
  • {{ fecha | date: "long" }}
  • {{ fecha | date: "dd/MM/YYYY" }}
  • {{ fecha | date: "EEEE dd" + ", MMMM YYYY" }}
  • {{ fecha | date: "long":"GMT-4" }}
  • {{ fecha | date: "long":"GMT-6":"fr" }}
